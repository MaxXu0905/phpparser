imagecolormatch

(PHP 4 >= 4.3.0)

imagecolormatch --  使一个图像中调色板版本的颜色与真彩色版本更能匹配

说明

bool imagecolormatch ( resource image1, resource image2)

警告

该函数目前尚无参考文档；仅参数列表可用。

image1 必须是真彩色图像，image2 必须是调色板图像，而且 image1 和 image2 的大小必须相同。

如果成功则返回 TRUE，失败则返回 FALSE。

参见 imagecreatetruecolor()。
