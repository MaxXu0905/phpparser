imagefilledellipse

(PHP 4 >= 4.0.6)

imagefilledellipse -- 画一椭圆并填充

说明

int imagefilledellipse ( resource image, int cx, int cy, int w, int h, int color)

imagefilledellipse() 在 image 所代表的图像中以 cx，cy（图像左上角为 0, 0）为中心画一个椭圆。w 和 h 分别指定了椭圆的宽和高。椭圆用 color 颜色填充。

注: 本函数添加于 PHP 4.0.6 并需要 GD 2.0.1 或更高版本。

参见 imagefilledarc()。
