file_put_contents

(PHP 5 CVS only)

file_put_contents -- 将一个字符串写入文件

说明

int file_put_contents ( string filename, string data [, int flags [, resource context]])

和依次调用 fopen()，fwrite() 以及 fclose() 功能一样。本函数返回写入文件的字节数。

flags 可以是 FILE_USE_INCLUDE_PATH 和/或 FILE_APPEND，但是使用 FILE_USE_INCLUDE_PATH 选项时要多加小心。

注: 该函数适用于二进制对象！

提示: 如果“fopen wrappers”已经被激活，则您在使用该函数的时候，可以把 URL 作为文件名来使用。请参阅 fopen() 获取怎样指定文件名的详细信息以及支持的 URL 协议的列表附录 J。

参见 fopen()，fwrite()，fclose() 和 file_get_contents()。
