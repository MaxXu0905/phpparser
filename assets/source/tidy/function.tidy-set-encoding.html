tidy_set_encoding

(no version information, might be only in CVS)

tidy_set_encoding --  Set the input/output character encoding for parsing markup.

Description

bool tidy_set_encoding ( string encoding)

Sets the encoding for input/output documents. 如果成功则返回 TRUE，失败则返回 FALSE。 Possible values for encoding are ascii, latin1, raw, utf8, iso2022, mac, win1252, utf16le, utf16be, utf16, big5, and shiftjis.

注: 该函数只在 Tidy 1.0 中存在。在 Tidy 2.0 中，该函数被移除。
