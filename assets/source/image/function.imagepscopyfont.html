imagepscopyfont

(PHP 3>= 3.0.9, PHP 4 )

imagepscopyfont --  拷贝一个已加载的字体以备更改

说明

int imagepscopyfont ( int fontindex)

如果要更改某字体例如扩展／精简，倾斜或改变字符编码矢量，但是又需要保留原有字体时使用本函数。注意要拷贝的字体必须是用 imagepsloadfont() 获得的，而不是一个已经是拷贝的字体。也可以在拷贝前修改。

如果使用本函数，必须按照相反的顺序释放用此方法取得的字体。否则你的脚本将死掉。

在一切正确的情况下，将返回一个有效的字体索引并可用于以后的用途。否则本函数返回 FALSE 并显示信息说明哪里错了。

参见 imagepsloadfont()。
