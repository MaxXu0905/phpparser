srand

(PHP 3, PHP 4 )

srand -- 播下随机数发生器种子

说明

void srand ( int seed)

用 seed 播下随机数发生器种子。

<?php
// seed with microseconds
function make_seed() {
    list($usec, $sec) = explode(' ', microtime());
    return (float) $sec + ((float) $usec * 100000);
}
srand(make_seed());
$randval = rand();
?>

注: 在 PHP 4.2.0 中，无需用函数 srand() 或 mt_srand()来搜寻随机数生成器，它将被自动完成。

参见 rand()，getrandmax() 和 mt_srand()。
