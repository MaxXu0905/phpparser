xml_get_current_line_number

(PHP 3>= 3.0.6, PHP 4 )

xml_get_current_line_number -- 获取 XML 解析器的当前行号

描述

int xml_get_current_line_number ( resource parser)

parser

一个指向要获取当前行号的 XML 解析器的指针。

如果 parser 参数没有指向一个合法的解析器，该函数将返回 FALSE，否则将返回指定解析器在其缓存中的当前行号。
