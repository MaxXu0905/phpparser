mysql_change_user

(PHP 3>= 3.0.13)

mysql_change_user --  改变活动连接中登录的用户

说明

int mysql_change_user ( string user, string password [, string database [, resource link_identifier]])

mysql_change_user() 改变当前活动连接中的登录用户，或者是可选的 link_identifier 参数所指定的连接。如果指明了数据库，将在改变用户后成为当前数据库。如果新用户和密码认证失败，当前连接的用户保持活动状态。如果成功则返回 TRUE，失败则返回 FALSE。

注: 本函数自 PHP 3.0.13 引进并需要 MySQL 3.23.3 或更高版本。不能用于 PHP 4。
