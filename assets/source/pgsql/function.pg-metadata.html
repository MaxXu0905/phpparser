pg_metadata --  获得表的元数据

说明

array pg_metadata ( resource connection, string table_name)

pg_metadata() 以数组形式返回 table_name 表的定义。如果出错则返回 FALSE。

例子 1. 取得表的元数据

<?php
    $dbconn = pg_connect("dbname=publisher") or die("Could not connect");
    $meta = pg_meta_data($dbconn,'authors');
    if (is_array ($meta)) {
        echo '<pre>';
        var_dump ($meta);
        echo '</pre>';
    }
?>
