disk_free_space

(PHP 4 >= 4.1.0)

disk_free_space -- 返回目录中的可用空间

说明

float disk_free_space ( string directory)

给出一个包含有一个目录的字符串，本函数将根据相应的文件系统或磁盘分区返回可用的字节数。

例子 1. disk_free_space() 例子

<?php
// $df 包含根目录下可用的字节数
$df = disk_free_space("/");
?>

注: 本函数不能作用于远程文件，被检查的文件必须通过服务器的文件系统访问。

参见 disk_total_space()。
