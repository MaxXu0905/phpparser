imagesx

(PHP 3, PHP 4 )

imagesx -- 取得图像宽度

说明

int imagesx ( resource image)

imagesx() 返回 image 所代表的图像的宽度。

例子 1. 使用 imagesx()

<?php

// create a 300*200 image
$img = imagecreate(300, 200);

echo imagesx($img); // 300

?>

参见 imagecreate()，getimagesize() 和 imagesy()。
