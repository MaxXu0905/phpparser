imagecolortransparent

(PHP 3, PHP 4 )

imagecolortransparent -- 将某个颜色定义为透明色

说明

int imagecolortransparent ( resource image [, int color])

imagecolortransparent() 将 image 图像中的透明色设定为 color。image 是 imagecreate() 返回的图像标识符，color 是 imagecolorallocate() 返回的颜色标识符。

注: 透明色是图像的一种属性，透明度不是颜色的属性。一旦设定了某个颜色为透明色，图像中之前画为该色的任何区域都成为透明的。

返回新透明色的标识符，如果省略 color 则返回当前透明色的标识符。
